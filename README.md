# eBay Wallet - Related Best Seller Automation

##  Project Overview
This project automates the Related Best Seller Products section
using Playwright.

##  Tech Stack
- Playwright
- JavaScript
- Page Object Model

##  How to Run

1. Install dependencies
   npm install

2. Run tests
   npx playwright test

##  Framework Structure
- tests/
- pages/
- utils/

##  Covered Scenarios
- Section visibility
- Max 6 products validation
- Out of stock exclusion
- Navigation validation
- Cross browser support

##  Reports
- Screenshots on failure
- Video recording on failure
